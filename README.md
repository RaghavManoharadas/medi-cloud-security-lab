# MediCloud Security Lab – Threat Analysis & Defense Simulation

## 📄 Full Technical Report
👉 [View Report](./report/medi-cloud-security-lab.pdf)

## Overview
This project simulates a real-world healthcare system (MediCloud) to analyze cybersecurity threats, design a secure lab environment, and evaluate defensive strategies against common cyber attacks.

## 🏥 System Context
MediCloud is a cloud-based healthcare platform handling:
- Electronic Health Records (EHR)
- IoT medical devices
- Patient data and clinical systems

## ⚠️ Threat Landscape
- Ransomware attacks targeting healthcare systems
- Phishing and social engineering attacks
- Web application vulnerabilities (SQL Injection, XSS)
- Cloud misconfiguration risks
- Insider threats and credential abuse

## 🔍 Key Findings
- Healthcare systems are high-value targets for attackers
- Weak configurations can expose sensitive patient data
- Lack of user awareness increases phishing risks
- Cloud misconfigurations lead to large-scale data exposure

## 🛡️ Security Controls Implemented
- Network segmentation and lab isolation
- Attack simulations (SQL Injection, MITM, ransomware)
- Secure architecture design for testing environments
- Monitoring and response strategies

## ⚙️ Security Strategies
- Multi-layered defense approach (defense-in-depth)
- Strong access control mechanisms
- Regular vulnerability assessments
- Employee awareness and training programs

## 🛠️ Tools & Techniques
- Virtualized lab environment
- Penetration testing tools
- Network monitoring techniques
- Threat simulation scenarios

## 📌 Conclusion
This project demonstrates how real-world organizations can identify, simulate, and mitigate cybersecurity threats using structured lab environments and proactive security strategies.
